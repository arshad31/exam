# Project 1: Registration Form

## Description
A simple registration form built using only HTML and CSS.

## Technologies Used
- HTML5  
- CSS3  

## Form Contents
- Full Name  
- Username  
- Email  
- Phone Number  
- Password  
- Confirm Password  
- Gender (Male, Female, Other)  

## How to Run
1. Download or clone the repository.
2. Open `index.html` in any web browser.
